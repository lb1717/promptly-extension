# Safari Web Extension (generated)

This folder is created by Apple's converter — not committed by default.

From the repo root on macOS:

```bash
xcrun safari-web-extension-converter . \
  --project-location safari/PromptlySafari \
  --app-name Promptly \
  --swift \
  --copy-resources \
  --force
```

Open the generated `.xcodeproj`, configure signing, build, and enable the extension in Safari → Settings → Extensions.

See [../browsers/README.md](../browsers/README.md) for full cross-browser setup.
